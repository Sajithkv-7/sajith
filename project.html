CAR DEALER PROJECT

Car Dealer Web Application

A full-stack car dealership management system built using Python, Django, HTML, CSS, and JavaScript.

Key Features:
  • User Authentication – secure registration, login, and profile management.
  • Payment Integration – Razorpay integration for secure transactions.
  • Customer Support Chatbot – real-time assistance within the platform.
  • Receipt Download – customers can download receipts after payments.
  • Star Rating System – users can rate and review services.
Role in the Project:

Designed and developed the platform using Django for backend logic, HTML/CSS for frontend styling, and integrated payment and chatbot features to improve user experience.
